# Resources Setup Instructions

## Images
Place your profile image at: `resources/img/jchen2@loyalty.com.jpg`

The current website expects a profile image file at this location. You can:
1. Add your own profile picture
2. Rename it to match the path above
3. Or update the image path in index.html

## Resume
Place your resume PDF at: `resources/pdf/final_resume_projects.pdf`

The resume link in the navigation menu points to this file.

## Publications
Add your research papers and publications to: `resources/publications/`

Example:
- `fairness-clinical-nlp.pdf`
- Other publication PDFs

These will be linked from the About section.
