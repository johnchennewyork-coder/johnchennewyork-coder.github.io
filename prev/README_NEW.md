# John Chen's Personal Website

This is my personal GitHub Pages website, built with HTML5, Bootstrap 4, and custom CSS.

## Features

- Responsive design for mobile, tablet, and desktop
- Typewriter effect for dynamic text animation
- Bootstrap 4 framework for responsive layout
- Google Fonts for custom typography
- Google Analytics integration
- Sections for About, Projects, Hackathons, Publications

## Structure

```
├── index.html                 # Main website
├── css/
│   └── custom/
│       └── style.css         # Custom styling
├── js/
│   └── typewriter/
│       └── typewriter.js     # Typewriter animation
├── resources/
│   ├── img/                  # Images and profile picture
│   ├── pdf/                  # Resume and documents
│   └── publications/         # Research papers
└── blog/                     # Blog section
```

## Getting Started

The website is automatically deployed via GitHub Pages from the `main` branch.

### Local Development

To view locally, simply open `index.html` in your web browser.
